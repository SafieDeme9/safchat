### Safchat
Safchat is a real-time messaging application built using React.js for the frontend, Python FastAPI for the backend, and Chat Engine for the chat functionality. It allows users to communicate with each other in real-time.

## Features
Real-time messaging: Users can send and receive messages in real-time.
User authentication: Users can sign up, log in, and authenticate themselves securely.
Chat rooms: Users can create and join different chat rooms for group conversations.

## Installation
To run the project locally, follow these steps:

Clone the repository:

```
git clone https://github.com/SafieDeme9/safchat
```
Navigate to the project directory:

```
cd safchat
```
Install dependencies:
```
npm install
```
Start the development server:
```
npm run dev
```
Your website will be running at http://localhost:5173/.

## Technologies Used
React.js: A JavaScript library for building user interfaces.
Python FastAPI: A modern, fast (high-performance), web framework for building APIs with Python.
Chat Engine: An API that makes it easy to build chat services.
## Usage
Sign up for an account or log in if you already have one.
Create or join chat rooms to start conversations with other users.
Send messages and interact with other users in real time.
